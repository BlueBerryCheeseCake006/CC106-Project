# CC106 Web Development Project

A responsive personal portfolio website built as the final project for **CC106 — Introduction to Web Development**.

## 🌐 Live Features

| Feature | Description |
|---|---|
| **Responsive Navbar** | Fixed navigation with scroll-shadow effect and mobile hamburger menu |
| **Hero Section** | Animated gradient headline with call-to-action buttons |
| **About Section** | Animated counters for stats (projects, study years, technologies) |
| **Skills Section** | Cards with animated progress bars triggered on scroll |
| **Projects Section** | Filterable project gallery (All / HTML+CSS / JavaScript) |
| **Contact Form** | Client-side validated contact form with success feedback |
| **Smooth Scroll** | CSS `scroll-behavior: smooth` for anchor navigation |
| **Scroll Animations** | Intersection Observer fade-in for all major sections |

## 🛠 Technologies Used

- **HTML5** — semantic structure (`<nav>`, `<section>`, `<footer>`, etc.)
- **CSS3** — custom properties, Flexbox, Grid, keyframe animations, media queries
- **JavaScript (ES6+)** — Intersection Observer API, DOM manipulation, form validation

## 📁 Project Structure

```
CC106-Project/
├── index.html   # Main HTML document
├── style.css    # All styles (variables, layout, responsive)
├── script.js    # Interactive behaviour
└── README.md    # Project documentation
```
